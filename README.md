## VueJS-FinalProject
### Required
1. Installed NodeJS
2. Create database name:smartphone
### Run app
1. Clone về, giải nén ở  thư mục htdocs
2. composer update
3. npm install
4. composer required laravel/passport (authentication)
5. php artisan migrate
6. php artisan db:seed
7. php artisan passport:install
8. php artisan serve
9. npm run dev
10. npm run prod
